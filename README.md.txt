# E-Commerce Sales Performance Analysis Dashboard

## Project Overview

This project analyzes e-commerce sales data using Microsoft Excel to identify business trends, customer behavior, product performance, and profitability.

An interactive dashboard was created using Pivot Tables, Pivot Charts, and Slicers to help stakeholders make data-driven decisions.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas
- Dashboard Design

---

## Dataset Information

Dataset Period:
2020 - 2024

Total Orders:
1050

Columns:

- Order ID
- Order Date
- Customer Name
- Region
- Category
- Product Name
- Sales
- Quantity
- Profit
- Payment Method

---

## Business Questions Answered

1. Which region generates the highest sales?

2. Which category generates the highest profit?

3. What are the monthly sales trends?

4. Which products generate the highest revenue?

5. Who are the top customers?

6. Are high sales categories also profitable?

---

## Key Insights

- Central region generated the highest sales.
- Furniture category generated the highest profit.
- October recorded the highest monthly sales.
- 2024 achieved the highest yearly sales growth.
- Clothing category requires profitability improvement.

---

## Business Recommendations

- Focus marketing efforts on high-performing regions.
- Promote profitable product categories.
- Improve pricing strategies for low-margin categories.
- Create loyalty programs for high-value customers.
- Use seasonal trends for inventory planning.
